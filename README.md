## About

RateFlix Movies is a responsive movie discovery and review platform built for movie lovers across languages, genres, and cultures.

Browse films, explore trailers, save favourites to your wishlist, and revisit stories worth remembering.

Built as a frontend web project using HTML, CSS, and JavaScript.

---

## Features

* Browse movies across multiple languages
* Search movies instantly
* Filter by:

  * Language
  * Genre
  * Theme
  * Actor
* Detailed movie modal with:

  * Poster
  * IMDb rating
  * Cast
  * Director
  * Producer
  * Budget
  * Box office
  * Songs
  * Streaming platform
* Watch official trailers
* Wishlist support using LocalStorage
* Add / remove movies from wishlist
* Remove saved movies directly from wishlist page
* About page with developer profile
* Contact page with success message after form submission
* Fully responsive design
* Dark cinematic UI inspired by modern streaming platforms

## Run Locally

Clone the repository:

```bash
git clone https://github.com/tanvipohankar-004/RateFlix-Movies.git
```

Open the project folder:

```bash
cd RateFlix-Movies
```

Run:

```bash
index.html
```

or launch using **Live Server** in VS Code.
